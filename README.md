# AG Host HVM Panel

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/aryan98765-osa/ag-host-hvm-panel.git
   cd ag-host-hvm-panel
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```

## Setup

1. Create a `.env` file in the root directory and add your environment variables:
   ```bash
   PORT=3000
   DB_URL=mongodb://localhost:27017/mydatabase
   ```
2. Start the application:
   ```bash
   npm start
   ```

## Usage

- Open your browser and go to `http://localhost:3000` to access the panel.
- Follow the prompts to configure your settings and start using the panel.
